The appear program is a small program that reads/interprets a poem. You can pass in .txt files of any poem to read. The appear program also supports instructions and styling as you proceed through the poem (see the instructions below). CONTROLS: Next line: DOWN, ENTER, SPACE Previous line: UP, BACKSPACE
You can run an instruction on a particular line of the poem by adding it after the text with a tilde and with the arguments separated by commas. For example, to make the line: this is blue text and this is lined text I would write something like: this is blue text and this is lined text~colorWhere,blue text,blue~underlineWhere,lined text INSTRUCTIONS // Style the current line
~color,[color]
~italic
~underline
~style,[styleKey],[styleValue]

// Style a range in the current line
~colorAt,[start],[end],[color]
~italicAt,[start],[end]
~underlineAt,[start],[end]
~styleAt,[start],[end],[styleKey],[styleValue]

// Style where the current line where it matches a regex pattern
~colorWhere,[pattern],[color]
~italicWhere,[pattern]
~underlineWhere,[pattern]
~styleWhere,[pattern],[styleKey],[styleValue]

// Modify a different line
~setLine,[lineNumber],[text]
~styleLine,[lineNumber],[styleKey],[styleValue]
~replaceLine,[lineNumber],[pattern],[replacement]
~clearLineStyle,[lineNumber]

// Global
~replace,[textHistory],[pattern],[replacement]
// Style all lines that match a pattern
~styleLineMatching,[pattern],[styleKey],[styleValue]
// Style a specified line where there is a match for the pattern
~styleLineWhere,[lineNumber],[pattern],[styleKey],[styleValue]
